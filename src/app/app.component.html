Easy Angular DnD // galczo5/angular-dnd

Easy Angular DnD is easy to use drag and drop library for Angular.
It provides maximum features with minimal dependencies. In fact angular is the only dependency of this package.

Easy Angular DnD is open source and free to use. It always will be.
For source code go to github page here.

Check out Examples for short demo of possible usage. In Tips and tricks you can find useful information that you can use to solve your problems.

Easy Angular DnD can be installed via npm or yarn.

NPM
npm install easy-angular-dnd

YARN
yarn add easy-angular-dnd
