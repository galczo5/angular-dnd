DragDirective

Drop directive should be used for defining elements

Name Type Default Description
[dndDrag] any null Defines data that will be dragged and dropped when drag ends
[disabled] boolean | string false Disables dragging on element
[group] string empty string Defines group for drop zones
(dragStarted) () => void
(dragEnded) () => void
